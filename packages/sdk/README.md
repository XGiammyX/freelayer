# @freelayer/sdk

**Status: placeholder client with honest status reporting, wired to the fail-closed core pipeline. No product operations exist yet.**

The public, stable surface for third-party tools building on FreeLayer: capsule creation/consumption, room object access, transport plugging — without exposing internals of `core`, `crypto`, or `privacy`.

The SDK cannot bypass the policy engine: third-party code gets the same privacy enforcement as first-party features.
